    Hat tip: why and how the hat matrix makes things easier 2021-03-05

    1. Intro
    2. The hat treatment
    3. Computing the covariance
    4. Conclusions

    Intro

    As most of my posts, this one is about linear modeling. I really believe that linear modeling is an under-appreciated discipline in today's machine learning mainstream. Also, as most of my posts, this one starts with a question in Cross Validated. I also believe that Cross Validated is under-appreciated, but I digress.

    So, the question is simple: how to show that the residual sum of squares (RSS) and the explained sum of squares (ESS) are independent random variables? I really wanted to answer this one, because it is fundamental, but from the start, it seemed quite clear that only an algebraic treatment would be feasible. This is due to quadratic forms that arise naturally in the problem, as you will see below.

    This was an important assumption in "The F-test and R²: an introduction from the ground up". I'll say it again to make the connection obvious here: the ratio of two independent Chi-square random variables is F-distributed.

    Let's state then what we have, shall we?

    y∼Normal⁡(βX,Σ=σ2I) y \sim \operatorname{Normal}(\beta X, \Sigma = \sigma^2 \mathbb I)\\ y∼Normal(βX,Σ=σ2I)

    This is our model, a plain, old, linear regression. Notice that we state it as a conditional distribution, where the expectation of the outcome is given by the linear combination of predictors βX\beta XβX and there is zero covariance between outcomes, which have a constant σ2\sigma^2σ2 variance attached. We could augment it, of course. But, from the treatment I'll expose below it'll become more obvious how it's actually a simple procedure.

    So, regarding the problem statement, here are RSS and ESS:

    {RSS=∑i=1n(yi−y^i)2ESS=∑i=1n(yi−yˉ)2 \begin{cases} \text{RSS} = \sum_{i=1}^n (y_i - \hat y_i)^2\\ \text{ESS} = \sum_{i=1}^n (y_i - \bar y)^2\\ \end{cases} {RSS=∑i=1n​(yi​−y^​i​)2ESS=∑i=1n​(yi​−yˉ​)2​

    Where y^i=βXi\hat y_i = \beta X_iy^​i​=βXi​ and yˉ=E(y)=1n∑i=1nyi\bar y = E(y) = \frac{1}{n}\sum_{i=1}^n y_iyˉ​=E(y)=n1​∑i=1n​yi​.

    So, how one goes on to show that RSS and ESS are uncorrelated?

    The hat treatment

    Even though the description of RSS and ESS are intuitive from their mathematical descriptions, let's take one step back (and add Algebra to the mix, making everything harder to read, though easier to solve).

    So, what's β\betaβ again? It's the solution to the problem, and it can be obtained in analytical form (I showed you how in: "Enter the Ridge: derivation and properties") :

    β=yXT(XXT)−1\beta = yX^T(XX^T)^{-1}β=yXT(XXT)−1

    With that we can retrieve the expected, or "fitted", values:

    y^=βX=yXT(XXT)−1X\hat y = \beta X = yX^T(XX^T)^{-1}Xy^​=βX=yXT(XXT)−1X

    Notice something peculiar? The fitted values y^\hat yy^​ can be written explicitly as a linear combination of the original values yyy. In fact, if we define H=XT(XXT)−1XH = X^T(XX^T)^{-1}XH=XT(XXT)−1X, then y^=yH\hat y = yHy^​=yH.

    We call this matrix HHH the hat matrix. It's a square matrix. It's very useful to demonstrate Gaussian linear regression properties. It also measure something called leverage: literally the leverage each original value in yyy has on the prediction of itself. Technically, it's simply the partial derivative ∂y^i/∂yi\partial \hat y_i/\partial y_i ∂y^​i​/∂yi​.

    Some neat properties can be attested outright:

    H2=(XT(XXT)−1X)(XT(XXT)−1X)=XT(XXT)−1(XXT)(XXT)−1X=XT(XXT)−1X=HH^2 = (X^T(XX^T)^{-1}X)(X^T(XX^T)^{-1}X)\\= X^T(XX^T)^{-1}(XX^T)(XX^T)^{-1}X\\=X^T(XX^T)^{-1}X=HH2=(XT(XXT)−1X)(XT(XXT)−1X)=XT(XXT)−1(XXT)(XXT)−1X=XT(XXT)−1X=H

    The hat matrix is idempotent!

    HT=(XT(XXT)−1X)T=XT(XXT)−TX=XT(XXT)−1X=HH^T = (X^T(XX^T)^{-1}X)^T = X^T(XX^T)^{-T}X = X^T(XX^T)^{-1}X = HHT=(XT(XXT)−1X)T=XT(XXT)−TX=XT(XXT)−1X=H

    The hat matrix is symmetric!

    So, as I promised you, using this makes things easier. Let's rewrite RSS and ESS. But first we must state them algebraically.

    RSS=ϵϵT=(y−y^)(y−y^)T=(y−yH)(y−yH)T=y(I−H)2yT=y(I−H)yT \text{RSS} = \epsilon\epsilon^T = (y - \hat y)(y - \hat y)^T = (y - yH)(y - yH)^T \\= y(I - H)^2y^T = y(I - H)y^T RSS=ϵϵT=(y−y^​)(y−y^​)T=(y−yH)(y−yH)T=y(I−H)2yT=y(I−H)yT

    Because (I−H)2=I2−2H+H2=I−2H+H=I−H(I - H)^2 = I^2 - 2H + H^2 = I - 2H + H = I - H(I−H)2=I2−2H+H2=I−2H+H=I−H.

    ESS=(y^−1μ)(y^−1μ)T=y^y^T−μ1Ty^−μy^1T+μ211TyHHTyT−nμ2−nμ2+nμ2=yHyT−nμ2 \text{ESS} = (\hat y - 1\mu)(\hat y - 1\mu)^T= \hat y\hat y^T - \mu 1^T \hat y - \mu \hat y 1^T + \mu^2 11^T\\ yHH^Ty^T - n \mu^2 - n \mu^2 + n \mu^2 = yHy^T - n \mu^2 ESS=(y^​−1μ)(y^​−1μ)T=y^​y^​T−μ1Ty^​−μy^​1T+μ211TyHHTyT−nμ2−nμ2+nμ2=yHyT−nμ2

    So there we have it, RSS=y(I−H)yT\text{RSS} = y(I - H)y^TRSS=y(I−H)yT and ESS=yHyT−nμ2\text{ESS} = yHy^T - n \mu^2ESS=yHyT−nμ2. That I−HI-HI−H term in the RSS might give you a hint they are uncorrelated, since ESS has simply a HHH. Now, how to prove they are uncorrelated?

    Computing the covariance

    The covariance is "simply" (haha) given by

    Cov⁡(RSS,ESS)=Cov⁡(y(I−H)yT,yHyT−nμ2)=Cov⁡(y(I−H)yT,yHyT)\operatorname{Cov}(\text{RSS},\text{ESS})=\operatorname{Cov}(y(I - H)y^T,yHy^T - n \mu^2)\\=\operatorname{Cov}(y(I - H)y^T,yHy^T)Cov(RSS,ESS)=Cov(y(I−H)yT,yHyT−nμ2)=Cov(y(I−H)yT,yHyT)

    Note that for symmetric matrices A,BA,BA,B:

    The covariance between quadratic forms[1] is given by:

    Cov⁡(xAxT,xBxT)=2tr⁡(AΣBΣ)+4μAΣBμT\operatorname{Cov}(xAx^T,xBx^T) = 2\operatorname{tr}(A \Sigma B \Sigma) + 4 \mu A \Sigma B \mu^TCov(xAxT,xBxT)=2tr(AΣBΣ)+4μAΣBμT

    Plugging A=(I−H)A = (I - H)A=(I−H) and B=HB = HB=H we retrieve:

    Cov⁡(y(I−H)yT,yHyT)=2tr⁡((I−H)ΣHΣ)+4μ(I−H)ΣHμT\operatorname{Cov}(y(I - H)y^T,yHy^T) = 2\operatorname{tr}((I - H) \Sigma H \Sigma) + 4 \mu (I - H) \Sigma H \mu^TCov(y(I−H)yT,yHyT)=2tr((I−H)ΣHΣ)+4μ(I−H)ΣHμT

    Using the fact that Σ=σ2I\Sigma = \sigma^2 IΣ=σ2I

    Cov⁡(y(I−H)yT,yHyT)=2σ4tr⁡((I−H)H)+4σ2μ(I−H)HμT=2σ4tr⁡(H−H2)+4σ2μ(H−H2)μT\operatorname{Cov}(y(I - H)y^T,yHy^T) = 2\sigma^4\operatorname{tr}((I - H) H) + 4 \sigma^2 \mu (I - H) H \mu^T\\ =2\sigma^4\operatorname{tr}(H - H^2) + 4 \sigma^2 \mu (H - H^2) \mu^TCov(y(I−H)yT,yHyT)=2σ4tr((I−H)H)+4σ2μ(I−H)HμT=2σ4tr(H−H2)+4σ2μ(H−H2)μT

    But H−H2=0H - H^2 = 0H−H2=0, so:

    Cov⁡(y(I−H)yT,yHyT)=0\operatorname{Cov}(y(I - H)y^T,yHy^T) = 0Cov(y(I−H)yT,yHyT)=0

    Conclusions

    To be frank, I am not even sure if a similar proof can be easily obtained without using the hat matrix. I hope that this post convinces you that the hat matrix helps a lot on the statistical treatment of linear regression. It also makes clear where different assumptions could be plugged in, e.g. Σ≠σ2I\Sigma \neq \sigma^2 IΣ​=σ2I.
